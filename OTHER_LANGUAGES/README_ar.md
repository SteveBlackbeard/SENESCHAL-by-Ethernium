# Seneschal

[![PyPI](https://img.shields.io/pypi/v/seneschal?label=pypi&color=blueviolet)](https://pypi.org/project/seneschal/)
[![License](https://img.shields.io/badge/license-MIT-green)](../LICENSE)

<div dir="rtl" markdown="1">

**Seneschal** هي طبقة تحكّم محلية أولًا تجعل العمل مع الذكاء الاصطناعي أرخص وأكثر أمانًا.

تقرّر أي سياق يُرسل، وما الذي يبقى محليًا، وما الذي يجب حجبه، وما إذا كانت المهمة تستحق نموذجًا أقوى. الهدف هو توفير حقيقي في الرموز (tokens): لصق أقل للمستودع بأكمله، ومحاولات إعادة أقل، ومطالبات أصغر، ونطاق أدوات أكثر أمانًا، وتسليم مهام أوضح.

> كان السنشال يدير موارد البيت ويأذن بالتصرفات نيابة عن سيده. وهذا بالضبط ما تفعله هذه الأداة: **توزّع الإنفاق** و**تأذن بالصلاحيات**.

## ماذا تفعل

- تقدير ميزانيات الرموز دون ارتباط بمزوّد بعينه
- تجميع سياق المستودع ضمن ميزانية صريحة
- التوصية بأرخص مسار نموذج يكفي لإنجاز المهمة
- حفظ لقطات من السياق لتفادي إعادة إرسال الملفات غير المتغيّرة
- اختيار السياق المجاور الأكثر صلة باستخدام قياس BM25 المعجمي
- فحص النصوص والملفات غير الموثوقة بحثًا عن حقن المطالبات والأسرار
- التحقّق من منح الصلاحيات بأقل امتياز، **الموقّعة بخوارزمية Ed25519**

## التثبيت

```bash
pip install seneschal
seneschal --help
```

إضافات اختيارية:

```bash
pip install "seneschal[measure]"    # مُجزّئ رموز مقيس (tiktoken)
pip install "seneschal[security]"   # منح موقّعة (cryptography)
```

## منح الصلاحيات الموقّعة

المنحة غير الموقّعة ليست سوى ملف JSON يستطيع أي عملية تعديله، بما في ذلك الوكيل الذي يُفترض أن تقيّده. مع توقيع Ed25519، يرفض الوسيط رفضًا مغلقًا أي منحة تفتقر إلى توقيع صالح، أو وُقّعت بمفتاح غير موثوق، أو عُدّلت بعد التوقيع (تصعيد الامتيازات)، أو انتهت صلاحيتها.

```bash
seneschal keygen
seneschal grant --sign --task-id RH-001 --capability read --allowed-path src/ --out grant.json
seneschal grant --grant-file grant.json --require-signed --task-id RH-001 --action read --path src/main.py
```

## ما ليست عليه

ليست بيئة معزولة ولا محرّك تنفيذ للوكلاء. هي تقرّر **هل** يُستدعى النموذج و**أي** سياق يُرسل إليه؛ ولا تتحكّم فيما يفعله ذلك النموذج أو الوكيل لاحقًا بالصلاحية الممنوحة.
